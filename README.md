# Code Critters — Desk Pets 🐾

Cute animated **pixel pets** that walk along a panel in your editor while you
code. Pick a cat, dog, or duck, give it a color, and let it wander. Drop a
treat and watch them run for it.

> 100% original pixel art, drawn live on a `<canvas>`no sprite assets, no
> tracking, no network access.

## Features

- 🐱 Three critters: **cat**, **dog**, **duck**,**Unique** , **Robot** each with several colors
- 🚶 Live walk / idle / chase animation along a floor
- 🦴 Click the floor (or run **Throw a Treat**) to drop a treat the pets chase
- 🐾 Spawn as many pets as you like — they persist across restarts
- 🎚️ Adjustable size: nano / small / medium / large

## Usage

1. Click the **paw icon** in the Activity Bar to open the **Code Critters** view.
2. Use the Command Palette (`Ctrl/Cmd+Shift+P`):
   - **Code Critters: Spawn a Pet** — pick a type and color
   - **Code Critters: Throw a Treat**
   - **Code Critters: Rename a pet**
   - **Code Critters: Remove a Pet** / **Remove All Pets**
3. Click anywhere on the floor to drop a treat 🦴
